# Landing Page

## Description
Multi-section landing page featuring a dynamic menu and smooth scroll effect.

## Features
- Dynamic Menu
- Smooth scroll to specific section of site
- Active state when menu item is clicked

## Technologies
- HTML
- CSS
- Vanilla JavaScript
